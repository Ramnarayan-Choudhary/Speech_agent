# Getting Started

1. Clone the repository:
   ```bash
git clone https://github.com/Ramnarayan-Choudhary/Speech_agent.git
cd Speech_agent
```

2. Create virtual environment:
   ```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

3. For Colab, open the notebook `notebooks/colab_indic_asr_training.ipynb` and run cells.

4. Use `example_configs` for language-specific configurations.
